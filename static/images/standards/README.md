# Standard Images

Place hardware standard images in this directory.

## Naming Convention
Use the standard ID as the filename:
- `iso4762.png` - Socket Head Cap Screw
- `iso4032.png` - Hex Nut
- `iso10642.png` - Countersunk Head Screw
- etc.

## Image Guidelines
- Preferred format: PNG with transparent background
- Recommended size: 200x200px or similar square aspect ratio
- Keep file sizes reasonable for web use
