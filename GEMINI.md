# Gridscribe - Gridfinity Label Generator

## Project Overview

This is a web application built with SvelteKit for creating and printing labels for the Gridfinity organization system. It allows users to generate labels for fasteners (screws, nuts, etc.) and general items, with options to customize the text, add QR codes, and include images of the hardware.

**Key Technologies:**

*   **Framework:** SvelteKit
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **UI Components:** bits-ui, tailwind-variants
*   **Testing:** Playwright, Vitest

## Building and Running

*   **Install dependencies:**
    ```bash
    pnpm install
    ```
*   **Run the development server:**
    ```bash
    pnpm run dev
    ```
*   **Build for production:**
    ```bash
    pnpm run build
    ```
*   **Run tests:**
    ```bash
    pnpm run test
    ```

## Development Conventions

*   **Code Style:** The project uses Prettier for code formatting and ESLint for linting. Configuration can be found in `.prettierrc` and `eslint.config.js`.
*   **Testing:** The project has both unit tests (run with Vitest) and end-to-end tests (run with Playwright). Test files are located alongside the source files (e.g., `src/routes/page.svelte.test.ts`) and in the `e2e` directory.
*   **Component Structure:** The application uses a component-based architecture, with UI components located in `src/lib/components/ui`.
*   **Data:** The application uses data from `src/lib/data/standards.ts` to populate the standards selection. This data is generated by the `scripts/build-all-standards.js` script.
