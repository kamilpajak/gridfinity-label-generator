# DIN Standard Images

This directory contains images for DIN standards in PNG format, organized in subdirectories by type:

- screws/   - Screw standards
- nuts/     - Nut standards  
- washers/  - Washer standards

Image naming convention:
- All lowercase
- Replace spaces with underscores
- Include "din_" prefix
- Include .png extension

Examples:
- screws/din_912.png
- nuts/din_934.png  
- washers/din_125.png

Images should be black and white technical drawings showing the standard's key characteristics.
