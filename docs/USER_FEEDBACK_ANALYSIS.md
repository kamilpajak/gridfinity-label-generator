# User Feedback Analysis

## Survey Period

April 29, 2025 - June 10, 2025 (10 responses)

## Overall Satisfaction

- Average interface rating: **4.4/5**
- Users consistently praise ease of use and the existence of the tool
- Main appreciation points: simplicity, standards lookup, beautiful images

## Feature Requests Summary

### 1. **Batch/Bulk Label Generation** (5 requests - 50%)

- Serial printing of multiple sizes in one task
- API for programmatic label generation
- Batch requests (e.g., M3 lengths 4,6,8,10,12)
- **Status**: ❌ Not implemented

### 2. **Material Type Selection** (3 requests - 30%)

- Support for material specifications (SS316, SS304, A2, A4, Brass)
- **Status**: ❌ Not implemented

### 3. **More Hardware Types** (3 requests - 30%)

- Threaded inserts (2 requests)
- Lock nut symbol
- More icons in general
- **Status**: ❌ Not implemented

### 4. **Advanced Customization** (2 requests - 20%)

- Baseline and gap adjustments
- Single-line text dimension controls
- Padding/margin controls
- **Status**: ❌ Not implemented

### 5. **Keyboard Navigation Improvements** (1 request - 10%)

- Auto-focus search field when dropdown opens
- Better TAB navigation
- **Status**: ✅ Implemented (v0.1.19 - keyboard navigation improvements)

### 6. **Label Height Adjustment** (1 request - 10%)

- Essential height control
- **Status**: ✅ Implemented (v0.1.19 - adjustable label height feature)

### 7. **Save/Load Settings** (1 request - 10%)

- Cookie-based settings persistence
- URL parameter support
- Export/import configuration
- **Status**: ❌ Not implemented

### 8. **Dark Mode** (1 request - 10%)

- **Status**: ❌ Not implemented

### 9. **ISO Standards Support** (1 request - 10%)

- More standards beyond DIN
- **Status**: ❌ Not implemented

### 10. **Simplified Screw Drawings** (1 request - 10%)

- Less detailed, more standardized icons:
  - Socket head
  - Low profile
  - Countersunk
  - Button head
- **Status**: ❌ Not implemented

### 11. **QR Code Positioning** (1 request - 10%)

- Center placement option for better scanning
- **Status**: ❌ Not implemented

### 12. **Smaller Width Support** (1 request - 10%)

- Half-width labels (0.5U)
- **Status**: ❌ Not implemented

## Key Issues Reported

### 1. **Label Size Limitations**

- Aspect ratio matching difficulties
- Independent tape size and printable area adjustment needed
- Some users work with non-standard sizes (9mm, 12x55mm)

### 2. **Printing Workflow**

- Poor margin control in browser print dialog
- Users forced to download and use external tools
- Repetitive work for multiple labels

## Recommendations Priority

### High Priority

1. **Batch Label Generation** - Most requested feature (50%)
2. **Material Type Selection** - Frequently requested (30%)
3. **Save/Load Settings** - Would significantly improve workflow

### Medium Priority

4. **More Hardware Types** (especially threaded inserts)
5. **Advanced Customization Options**
6. **Dark Mode**

### Low Priority

7. **ISO Standards Support**
8. **Simplified Icon Sets**
9. **QR Code Positioning Options**

## Already Implemented Features

Based on the changelog, these requested features have been addressed:

- ✅ Keyboard navigation improvements (v0.1.19)
- ✅ Adjustable label height (v0.1.19)
- ✅ Adjustable text size (v0.1.19)
- ✅ Improved label generation for different sizes (v0.1.19)
